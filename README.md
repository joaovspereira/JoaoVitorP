# Vehicle Listings Dashboard

Este projeto apresenta um aplicativo web desenvolvido com Streamlit para explorar dados de anúncios de venda de carros nos Estados Unidos.

## Funcionalidades
- Exibição de uma prévia do conjunto de dados
- Geração de histograma interativo
- Geração de gráfico de dispersão interativo
- Dashboard online com Streamlit
- Deploy em nuvem com Render

## Tecnologias utilizadas
- Python
- pandas
- plotly
- streamlit

## Estrutura do projeto
- `app.py`: aplicação principal
- `vehicles_us.csv`: conjunto de dados
- `notebooks/EDA.ipynb`: análise exploratória
- `.streamlit/config.toml`: configuração do Streamlit
- `requirements.txt`: dependências

## Como executar localmente

streamlit run app.py
